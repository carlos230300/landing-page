## Como executar localmente

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/teste-six.git

2. Entre na pasta do projeto:

cd teste/teste-six

3. Instale as dependências:

npm install
# ou
yarn
# ou
pnpm install

4. Rode o servidor de desenvolvimento:

npm run dev
# ou
yarn dev
# ou
pnpm dev

5. Abra o navegador:

http://localhost:3000
